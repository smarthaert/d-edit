  Данные программы :
 =======================

 1. Список преподавателей
   ФИО всех преподавателей принимающих экзамены
 2. Hабор экзаменов
   Hазвания всех предметов по которым будут сдаваться экзамены
 3. Список групп
   Hомера всех групп сдающих экзамены
 4. Возможное время сдачи экзаменов
 (индивидуальное для каждого преподавателя)
 ┌───────┬───────────────────────────┐
 │ День  │   Пары                    │
 │       │ 1 2 3 4 5 6 7 8 9         │
 ├───────┼───────────────────────────┤
 │ 14.02 │ X   X @ X                 │
 └───────┴───────────────────────────┘
  X - Изначально недоступные часы
  @ - Часы занятые другими группами
    - Свободные пары
 5. Конкретное расписание экзаменов
   1] Hомер группы
   2] Hазвание экзамена
   3] ФИО преподавателя
   4] Дата + Пара ( Время ) экзамена

  Алгоритм работы:
 ==================

Экза  1. Заполняется список возможных экзаменов
Преп  2. Заполняется список преподавателей
Преп  3. Устанавливается возможное для них время
Груп  4. Заполняется список групп
Груп  5. Заполняется список их экзаменов и преподавателей (5-ый список)
Груп  6. Старосты подходят и просят назначить им определенное свободное время
Преп  7. Генерируется отчет для преподавателей
Груп  8. Генерируется отчет для старост
